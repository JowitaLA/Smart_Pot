# Opis

# Co umożliwia inteligentna doniczka??
Podlewania ze zbiornikiem (przy smart to uzupełnianie zbiornika z wodą)
Zasłonięcie/odsłonięcie inteligentnej rolety bądź zaświecenie sztucznej lampy (zależnie od pory dnia)
Generowanie wykresów
Tryb SMART (uzupełnianie wody i włączanie sztucznej lampy/zasywanie i odsuwanie rolety)
